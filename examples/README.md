# Examples

These are tiny working specimens, not production content.

They show a full mini chain:

1. raw source in `examples/raw/processed/...`
2. source-note in `examples/wiki/mini-source-note.md`
3. concept in `examples/wiki/mini-concept.md`
4. analysis in `examples/wiki/mini-analysis.md`
5. comparison in `examples/wiki/mini-comparison.md`
6. index + log style in `examples/wiki/index.md` and `examples/wiki/mini-log.md`

Use this folder to learn the method quickly, then apply the same pattern in real `raw/` and `wiki/`.
